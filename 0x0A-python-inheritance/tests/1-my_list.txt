============================
1-my_list module
============================


`MyList` class
-----------------------------------
Initiates MyList class that derrived from a list class,
and extend it by a print_sorted method.

>>> MyList = __import__('1-my_list').MyList

>>> newlist = MyList()

>>> newlist = MyList('str')

>>> newlist = MyList([1, 2, 3])

>>> newlist = MyList(999)
Traceback (most recent call last):
TypeError: 'int' object is not iterable

>>> newlist = MyList([2, 3], [5, 6])
Traceback (most recent call last):
TypeError: list expected at most 1 argument, got 2


`print_sorted` method
-----------------------------------
print a sorted version of the list.

>>> newlist = MyList()
>>> newlist.print_sorted()
[]

>>> newlist.append(1)
>>> newlist.append(4)
>>> newlist.append(3)
>>> newlist.append(7)
>>> newlist.append(0)
>>> newlist.append(-5)
>>> print(newlist)
[1, 4, 3, 7, 0, -5]
>>> newlist.print_sorted()
[-5, 0, 1, 3, 4, 7]

>>> newlist.clear()

>>> newlist.extend([3.5, 2.5, 5.1, 2.3])

>>> print(newlist)
[3.5, 2.5, 5.1, 2.3]

>>> newlist.print_sorted()
[2.3, 2.5, 3.5, 5.1]

>>> newlist.clear()

>>> newlist.extend([-2, -6, -3, -1])
>>> newlist.print_sorted()
[-6, -3, -2, -1]

>>> newlist.clear()
>>> print(newlist)
[]
